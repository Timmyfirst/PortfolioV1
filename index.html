Site de e-commerce

Site de e-commerce permettant la consultation des articles repertoriés dans une base donnée, mise à jour automatique de celui-ci lorsque des articles sont ajoutés à la base. Permet également l'inscription au site et la constitution d'un panier virtuel. De plus une fois inscris des commentaires peuvent être émis sur la pgae de description d'un article.

  • Date: Novembre-Décembre 2015
  • Service: Développement web

Application lourde de gestion

Application lourde compatible windows permettant la constitution et la gérance de rendez-vous commerciaux, ainsi que la gérance des clients et des prospects. Cette application est reliée à une base de donnée où sont stockés les rendez-vous, les clients et les prospects.

  • Date: Novembre-Décembre 2015
  • Service: Développement c#

  • Service: Développement web

Application favoris youtube

Application lourde compatible windows permettant la constitution d'une bibliothèque de liens youtubes stockés dans une base de donnée. Ces liens peuvent être ajoutés et supprimés, un texte peut leur être associé. Ils sont organisés par catégories qui peuvent aussi également être gérées. L'application permet aussi le visionnage des vidéos associées aux liens.

  • Date: mars 2016
  • Service: Développement c#
